### 桶排序

   * 桶排序，即将数据划分为不同的“桶”进行排序。具体操作为：1. 分桶 2. 桶内排序 3. 合并桶

   * 分桶：根据数据范围（最大最小值）确定好桶的数量

   * 桶内排序：每个桶内独立进行排序（如快速排序），使得桶内有序

   * 合并桶：将每个桶内的有序数据逐次追加至结果中，并保持数据有序，最后得到完整的有序数据

---

### 代码实现

    def bucket_sort(data):
        length = len(data)
        if length <= 1:
            return
        # 桶的个数
        bucket_num = max(data) // 10 + 1
        # 创建n个桶
        bucket_list = [[] for i in range(0, bucket_num)]
        # 把每个元素装入对应的桶内
        for j in range(0, length):
            num = data[j] // 10
            bucket_list[num].append(data[j])
        result = []
        # 桶内排序，依次拼接
        for i in range(0, bucket_num):
            bucket_list[i].sort()
            result.extend(bucket_list[i])
        return result

### 性能分析

   * 时间复杂度：O(n)
   * 空间复杂度：O(k * n)
