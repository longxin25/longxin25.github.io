## 数据结构与算法笔记

### 写在前面

---

  数据结构与算法是程序员的一门基础必修课，有着不可或缺的重要性。虽然在现实的业务开发中，我们往往使用的是封装好的接口以及函数，并不需要关注其底层实现。但是在一些场景，比如海量数据存储，此时采用不同的数据结构将会带来天差地别的影响。当我们很好地掌握了数据结构以及算法之后，我们将能够在具体的业务场景中快速、合理地选择相应的算法并进行优化，设计出高质量的程序。
  
  本仓库为本人学习数据结构和算法的笔记整理，内容来源自互联网中数据结构与算法相关教程及书籍，具体包括但不仅限于以下内容：
  
  [王争  《数据结构与算法之美》](https://github.com/wangzheng0822/algo)
  
  [Dairongpeng  algorithm-note](https://github.com/Dairongpeng/algorithm-note)
  
  [pwstrick  一句话算法](https://github.com/pwstrick/daily/blob/master/article/one/one.md)
  
  [CyC2018  CS-Notes-算法](https://github.com/CyC2018/CS-Notes/blob/master/notes/%E7%AE%97%E6%B3%95%20-%20%E7%9B%AE%E5%BD%95.md)
  
  本笔记以循序渐进的方式对数据结构和算法相关的知识进行了梳理，并且贯穿了一些实际应用场景，包括但不仅限于底层设计、工业场景等，尝试剖析其底层设计思想和实现过程。目前整理了绝大部分基础内容，并进行了修正和简化，后续会通过不断更新来力求内容的全面和丰富。仓库内的参考代码采用 Python3 实现，并且经过了随机运行结果检验。如果您有任何问题或者建议，欢迎提交 issue 讨论。
