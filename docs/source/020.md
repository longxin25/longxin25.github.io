### 数组

  * 数组属于线性表，用一组连续的内存空间来存储各个数据，支持根据下标实现的随机访问，但其容量有限（一般取决于初始化时开辟的内存空间大小）。

---

### 代码实现

    class Array:

          def __init__(self, capacity: int) -> None:
              # 数据
              self.__data = []
              # 容量
              self.__capacity = capacity

          def find(self, index: int) -> object:
              try:
                  return self.__data[index]
              except IndexError:
                  return None

          def delete(self, index: int) -> bool:
              try:
                  self.__data.pop(index)
                  return True
              except IndexError:
                  return False

          def insert(self, index: int, value: int) -> bool:
              # 溢出
              if len(self.__data) >= self.__capacity:
                  return False
              else:
                  self.__data.insert(index, value)
                  return True

          def print_all(self) -> None:
              for item in self.__data:
                  print(item)
                  
---

### 性能分析

  * 由于其内存连续的特点，在随机访问元素时能够达到 O(1) 的高效时间复杂度，但在进行插入和删除操作时需大量移动元素，此时时间复杂度为 O(n)。

---

### 特点

  1. 内存连续
  2. 不可自动扩容
  3. 随机访问效率高，插入和删除效率低
